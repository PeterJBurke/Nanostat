Instructions:

On bootup, blue LED flashes until setup complete.

To run a sweep, goto home page and click button for sweep type. (See below for sweep type explanations.) During sweep, blue LED flashes.

To set sweep parameters, goto setup page.

To bias at a specific point and monitor voltages, goto panel page.

Calibration:

To calibrate, run "Cal" with WE disconnected from RE/CE. (Short RE/CE). Calibrations saved until reboot. Rough calibration parameters hard coded.

Sweep modes:

CV = Cyclic voltammetry.
Bioanalytical Systems, Inc. tutorial on CV

NPV = Normal pulsed voltammetry.
Bioanalytical Systems, Inc. tutorial on NPV

SWV = Square wave voltammetry.
Bioanalytical Systems, Inc. tutorial on SWV

CA = Chronoamperometry.
Bioanalytical Systems, Inc. tutorial on CA

DC = Fixed DC bias.
IV = IV curve.
Cal = calibrate ESP32Stat.
